# vscode-conda-cheatsheet
Conda Cheatsheet, `Ctrl+Shift+P` / `Cmd+Shift+P` / `F1` + `Conda Cheatsheet` command.

## Features
- `Conda Cheatsheet: PDF` to open PDF file inside editor
- `Conda Cheatsheet: Website` to open website inside editor
